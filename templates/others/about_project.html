About The Project

Himalya Kart is a comprehensive e-commerce platform designed to provide a seamless shopping experience. This project was designed with a focus on user-friendly design and robust functionality.

This project is created by samyam sapkota. It uses technlogies like HTML, CSS, JavaScript, and Bootstrap for the frontend. The backend is powered by Django, which provides a secure and scalable framework for building web applications and uses dbsqlite as the database. You can get the source code for this project on my GitHub.

Project Features

  • Responsive design that works seamlessly across desktop, tablet, and mobile devices
  • User authentication with secure login, registration, and password recovery
  • Product catalog with categories, filters
  • Shopping cart with add, remove, and quantity adjustment capabilities
  • Secure checkout process with stripes payment gateway (sandbox environment)
  • Detailed order history
  • Admin dashboard for product, order, and user management
  • Inventory management system with stock level alerts
  • Email notifications for order confirmations
  • Form for leaving queries and contacting
